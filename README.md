# Monkey

An interpreted language written in Go

## Dependencies

+ `direnv`
+ `just` (not necessary)
+ Go 1.21+
+ ginkgo (if you want to run local unit tests)
+ golangci-lint (if you want to do local linting)

## Conventions

+ Identifiers only consist of alphabet letters or underscore
